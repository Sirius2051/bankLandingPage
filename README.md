## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

You need to have installed Node.js on your computer

### Installation

1. Clone the repo:
```sh
   git clone https://github.com/Sirius2051/bankLandingPage.git
   ```
2. Install dependencies:
```sh
   npm install
   ```
3. Run the project:
```sh
   npm start
   ```
4. Done. :)
